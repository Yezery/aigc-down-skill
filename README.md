# AIGC-Down-Skill

中文学术论文 AIGC 检测率降低工具，专为专科、本科毕业论文设计。

## 核心功能

- **智能解析检测报告**：支持导入知网、维普等平台的 HTML 格式 AIGC 检查报告，自动提取高风险标注段落
- **针对性改写**：基于真实论文改写实验归纳的规律，将 AIGC 率从 >50% 有效降至 11% 左右
- **学术规范保障**：改写过程中保留核心论证逻辑，维持学术语域，不新增事实、案例、数据

## 使用方式

### 1. 提供检测报告

将 AIGC 检查报告文件（.html 格式，推荐）发送给 Agent 助手，工具会自动解析并提取 AIGC 率高的标注段落。

### 2. 风险识别

输出风险识别报告，列出：
- 高风险段落位置
- 命中的 AI 写作模式类型
- 硬约束违反情况

### 3. 确认改写

用户确认后，输出人工化改写版本，并整理为 .txt 文件。

## AI 写作模式识别

本工具可识别并修复以下典型 AI 写作模式：

### 内容层面
- 「依据/基于XX理论」起笔模式
- 「此案例印证了」段末套路
- 「首先/其次/再次」编号逻辑
- 被动分析套话
- 模板化问题陈述
- 高度对称的并列句式
- 画蛇添足的总结句
- 模糊归因（「专家认为」无出处）
- 泛化结论与「具有重要意义」结尾

### 语言层面
- AI 高频词汇（深刻揭示了、具有重要意义、综合运用等）
- 系动词替换（作为...的重要载体、扮演着...的角色）
- 过度对仗的排比结构
- 填充短语与过度限定

### 风格层面
- 破折号滥用或完全不用
- 加粗滥用

## 改写质量保障

改写完成后进行多维度评估：

| 维度 | 评估标准 |
|------|----------|
| 直接性 | 直接陈述，不绕圈 |
| 节奏 | 长短句交替，有变化 |
| 真实性 | 像真实学者表达 |
| 信息密度 | 每句话承载信息 |
| 学术规范 | 归因具体，限定合理 |
| 抗检测性 | 模式规律性被打破 |

## 适用范围

- **文档类型**：专科、本科毕业论文
- **语言**：中文学术写作
- **报告格式**：HTML 格式（推荐）的 AIGC 检查报告

## 使用提示

1. 提供完整的 HTML 检测报告可获得更精准的段落定位
2. 改写会保留原文核心意思，不改变论证逻辑与学术观点
3. 如需特定风格调整，可在改写前说明偏好
